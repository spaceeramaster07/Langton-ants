What is this?

Langton's ant is a mathematical cellular automaton. The ant reads the color on the floor, rotates, paints a new color over it, and takes a step!

Custom Rules

Prototype your own algorithms using the Custom Rule box on the right panel!

Build an algorithmic dictionary mapping sequence. The integer Key (e.g. "0:" or "4:") represents the specific color the ant currently sees beneath its feet. The writeColor property mathematically defines what new color the ant will paint over that cell!

{
  0: { writeColor: 4, move: 'R' },
  4: { writeColor: 0, move: 'L' }
}

Once structured, type a name and hit Save Preset. Your algorithm will lock into the dropdown and live in your browser memory endlessly!
